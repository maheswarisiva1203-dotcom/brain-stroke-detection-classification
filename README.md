# brain stroke detection and classification system
Brain Stroke Detection Project

Steps:

1. Install requirements:
   pip install -r requirements.txt

2. Prepare dataset folder:
   dataset/
     ├── No_Stroke/
     ├── Ischemic/
     ├── Hemorrhagic/

3. Train model:
   python train_model.py

4. Run app:
   python app.py

5. Open browser:
   http://127.0.0.1:5000
